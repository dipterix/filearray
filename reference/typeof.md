# The type of a file array (extended)

The type of a file array (extended)

## Usage

``` r
typeof(x)

# S4 method for class 'FileArray'
typeof(x)

# S4 method for class 'FileArrayProxy'
typeof(x)
```

## Arguments

- x:

  any file array

## Value

A character string. The possible values are `"double"`, `"integer"`,
`"logical"`, and `"raw"`
